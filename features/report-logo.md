# Report Logo (Premium)

Add a custom logo (such as your school or district logo) to the header of generated student reports.

## Requirements

- **Valid License**: ✅ Required
- **Must be GradeBook**: ✅ Required
- **Premium Plus**: ✅ Required

## Accessing the Feature

1. Open your GradeBook
2. Click **Extensions → GradeBook → Utilities → Report Logo**
3. The Report Logo sidebar will open

---

## Menu Cards

### Logo Preview

Shows your currently uploaded logo. If no logo is set, this area is blank.

If a logo is uploaded, you can click **Delete Logo** to remove it.

### Upload Logo

Click **Upload Logo** to select and upload an image file from your computer. Supported formats include PNG and JPEG.

### Logo Size

A dropdown to select how large the logo appears on reports:
- **Small**
- **Medium**
- **Large**
- **Extra Large**

### Logo Display Settings

Controls where and how the logo is used:

**Logo Scope Control**
- **Local (This GradeBook only)**: The logo is used only for reports generated from this specific GradeBook
- **Global (All GradeBooks)**: The logo is used for reports from all your GradeBooks

**Include Logo on Reports**
- Check this box to include the logo on generated reports
- Uncheck to hide the logo without deleting it

### Generate Sample Report

Click **Generate Sample Report** to create a test report showing how your logo will appear. This lets you check the size and appearance before generating actual student reports.

---

## Tips

- Use a high-resolution image for best print quality
- PNG files with transparent backgrounds work well
- Test with a sample report before generating all reports
