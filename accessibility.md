# Accessibility Settings

GradeBook includes comprehensive accessibility features that are fully compliant with **WCAG 2.1 AA standards**.

## Documentation Site Accessibility

The GradeBook documentation site includes accessibility features to ensure all users can comfortably read and navigate the documentation.

### Theme Toggle

A floating accessibility button (â™¿) is available in the bottom-right corner of every documentation page, allowing you to toggle between light and dark visual themes:

- **Light**: High contrast with black text on white background
- **Dark**: High contrast with white text on black background

Click the button to switch between themes. Your preference is automatically saved and will persist across all documentation pages and future visits.

## WCAG 2.1 AA Compliance

The GradeBook's accessibility features are designed to meet or exceed WCAG 2.1 AA guidelines, including:

- **Color Contrast**: All text and interface elements maintain sufficient contrast ratios in both themes
- **Screen Reader Support**: Full compatibility with screen readers through proper ARIA labels
- **Keyboard Navigation**: The toggle button is accessible via keyboard input
- **Focus Indicators**: Clear visual focus indicators for the toggle button
- **Semantic HTML**: Proper use of headings, landmarks, and form controls

## Additional Features

- Theme preference persists across all documentation pages and sessions
- The toggle button provides appropriate accessibility labels for screen readers
