# Accessibility Settings

GradeBook includes comprehensive accessibility features that are fully compliant with **WCAG 2.1 AA standards**.

## Accessing the Accessibility Settings

To access the accessibility settings:

1. Look for the floating accessibility button in the bottom-right corner of any GradeBook menu 
2. The button displays a universal access icon
3. Click the button to open the Accessibility Settings

## Available Settings

### Contrast Mode

The GradeBook provides three contrast modes to accommodate different visual needs:

- **Default**: Standard colors and contrast levels
- **Dark**: High contrast with white text on a black background
- **Light**: High contrast with black text on a white background

To change the contrast mode:
1. Open the Accessibility Settings modal
2. In the "Contrast Mode" section, select your preferred option
3. Changes are applied immediately and saved automatically

### Text Size

Four text size options are available to improve readability:

- **Default (100%)**: Normal text size
- **Medium (110%)**: Slightly larger text
- **Large (125%)**: Significantly larger text
- **XL (150%)**: Maximum text size

To change the text size:
1. Open the Accessibility Settings modal
2. In the "Text Size" section, select your preferred option
3. Changes are applied immediately and saved automatically

## WCAG 2.1 AA Compliance

The GradeBook's accessibility features are designed to meet or exceed WCAG 2.1 AA guidelines, including:

- **Color Contrast**: All text and interface elements maintain sufficient contrast ratios in all modes
- **Screen Reader Support**: Full compatibility with screen readers through proper ARIA labels and live regions
- **Keyboard Navigation**: All controls are accessible via keyboard input
- **Focus Indicators**: Clear visual focus indicators for all interactive elements
- **Semantic HTML**: Proper use of headings, landmarks, and form controls

## Additional Features

- Settings persist across all GradeBook menus and sessions
- Changes are announced to screen readers when made
- Tooltips provide additional guidance for each setting
- The floating button updates to show current settings in its accessibility label
